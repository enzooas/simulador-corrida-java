# 🏎️ Simulador de Corrida em Java

## 📌 Descrição

Este projeto é um simulador de corrida desenvolvido em Java utilizando Programação Orientada a Objetos (POO).

Dois carros competem em uma pista, onde suas velocidades são geradas aleatoriamente a cada rodada, até que um deles alcance a linha de chegada.

---

## 🎮 Como funciona

* Dois carros (ex: Ferrari e McLaren) participam da corrida
* A cada ciclo:

  * Uma velocidade aleatória é gerada para cada carro
  * A distância percorrida é atualizada
* A corrida continua até que um dos carros atinja o final da pista
* Ao final, um relatório com o resultado é exibido

---

## 🛠 Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)
* Threads (`Thread.sleep`) para simulação de tempo
* Estruturas de repetição e lógica condicional
